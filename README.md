# Green Alpine Peaks

**Green Alpine Peaks** — учебное Spring Boot приложение, моделирующее платформу проживания и работы в Альпах: фермы, регионы и возможности экологичного образа жизни в горах.

Проект демонстрирует базовую архитектуру REST API с использованием Spring Boot.

---

## Предметная область

Проект описывает фермы в Альпах, которые:

* расположены в разных регионах,
* предоставляют проживание,
* могут быть отфильтрованы по региону.

Ключевая доменная сущность проекта — Farm.

---

## Архитектура

Приложение реализовано с классической слоистой архитектурой:

```
Controller → Service → Repository → Database
```

Используются:

* DTO для API-ответов
* Mapper для преобразования Entity → DTO

---

## REST API

На текущем этапе реализованы следующие HTTP-эндпоинты:

### Получить все фермы

```http
GET /api/farms
```

**Описание:**
Возвращает список всех доступных ферм.

---

### Получить фермы по региону

```http
GET /api/farms/filter?region=Alps
```

**Параметры запроса:**

* `region` — регион, по которому выполняется фильтрация

---

### Получить ферму по идентификатору

```http
GET /api/farms/{id}
```

**Описание:**
Возвращает одну ферму по её ID.

## Сборка и запуск проекта

### Сборка

```bash
./mvnw clean install
```

### Запуск

```bash
./mvnw spring-boot:run
```

После запуска приложение будет доступно по адресу:

```
http://localhost:8080
```
